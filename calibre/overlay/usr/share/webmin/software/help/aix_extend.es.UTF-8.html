¿ EXTENDER los sistemas de ficheros si se necesita espacio ?
Indica si desea que el sistema agrande los sistemas de ficheros si se necesita más espacio durante el procesamiento.

SÍ indica al sistema que agrande los sistemas de ficheros lo necesario. Una vez el sistema de ficheros se ha agrandado, no se puede reducir, deberá eliminar el sistema de ficheros entero. Si dispone de suficiente espacio en disco duro, elija SÍ. Para previsualizar la cantidad de espacio necesario, diga SÍ a la opción "¿ Sólo PREVISUALIZAR ?".

NO le indica al sistema que NO agrande los sistemas de ficheros. Si dispone de espacio en disco limitad, elija NO. Si elige NO y el sistea se queda sin espacio en disco durante el procesamiento, se mostrará un mensaje del sistema. El mensaje indica cuánto espacio en disco es necesario y qué fichero del sistema necesita dicho espacio. El sistema no instalará el producto que necesite el espacio adicional, pero continuará el proceso de instalación de otros productos. En algunos caso se podría marcar una actualización como ROTA se no hay suficiente espacio en disco y el proceso de limpieza no se puede completar.
