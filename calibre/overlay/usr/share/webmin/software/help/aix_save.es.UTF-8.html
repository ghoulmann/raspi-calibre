¿ GUARDAR los ficheros sustituidos ?
Indica si quiere guardar los ficheros de software que hayan sido sustituidos durante la instalación.

SI indica al sistema que guarde las versiones previas del software que está sustituyendo. En caso de una instalación fallida, elegir SI permite la ejecución del procedimiento de limpieza para restaurar la versión previa del software. Si elige NO en "¿ COMPROMETER software ?", entonces debe elegir SÍ en "¿ GUARDAR los ficheros sustituidos ?". Elegir SÍ necesitará de espacio adicional para guardar los ficheros sustituidos. Elegir SÍ también necesita de más tiempo de instalación por el trabajo extra de guardar los ficheros.

NO indica al sistema que NO guarde las versiones previas del software mientras que se sustituye. En caso de una instalación fallida la versión previa del software NO se puede restaurar y se marca como ROTA. Si esto ocurre deberá reinstalar la actualización del software.
