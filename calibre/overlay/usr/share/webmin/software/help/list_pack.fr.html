Fichiers des composants
Cette page liste tous les fichiers contenus dans un composant. Pour chaque fichier les informations suivantes sont affichées :
Chemin
Le chemin absolu de ce fichier. Si c'est un lien symbolique, la cible de ce lien va être incluse au chemin. Cliquez sur le chemin d'accès pour avoir plus d'informations sur le fichier.
Propriétaire
L'utilisateur Unix qui devrait possèder ce fichier.
Groupe
Le groupe Unix qui devrait possèder ce fichier.
Type
Le type de fichier ou de répertoire dont il s'agit.
Taille
La taille que le fichier devrait avoir.
État
A moins que le fichier soit censé être modifiable, cette colonne affiche toutes les anomalies entre la taille, le propriétaire et le 'checksum' du fichier et leurs valeurs courantes.
