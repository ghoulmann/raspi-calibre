Port NFS
El valor numèric del port a connectar en el servidor NFS.
Si el número de port és 0 (el valor per defecte) llavors consulta el mapejador de ports del host remot per saber el número de port. Si el dimoni NFS del host remot no està registrat amb el seu mapejador de ports, s'utilitza en lloc seu el número de port estàndard de NFS 2049.

opció de muntatge: port
defecte: 0
