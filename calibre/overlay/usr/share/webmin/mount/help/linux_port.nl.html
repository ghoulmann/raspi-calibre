NFS Poort
De numerieke waarde van de poort waarop een NFS server verbinding maakt.
Indien het poort nummer 0 is (standaard) zal de remote host portmapper een vrije poort opvragen om te gebruiken. Als de remote host NFS deamon niet is geregistreerd met zijn portmapper zal de standaard poort 2049 worden gebruikt.

mount optie: poort
Standaard: 0
