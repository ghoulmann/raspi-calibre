Port NFS
La valeur numérique du port utilisé pour se connecter sur le serveur NFS.
Si cette valeur vaut 0 (par défaut) alors le gestionnaire de ports de l'hôte distant (portmapper) est interrogé pour connaître le numéro de port à utiliser. Si le démon NFS de l'hôte distant n'est pas enregistré dans le gestionnaire de ports, la valeur standard pour le port NFS, 2049, est utilisée à la place.

Option de mount : port
valeur par défaut: 0
