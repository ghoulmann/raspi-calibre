Deze pagina toont opties voor een enkele host, hosts waarvan de namen overeenkomen met sommige patronen, of alle hosts. Sommige minder duidelijk opties zijn :
Login als gebruiker
Normaal gesproken indien er geen gebruikersnaam is opgegeven op de ssh opdracht regel, word de naam van de huidige gebruiker gebruikt om in te loggen bij de remote server. Echter deze optie kan worden gebruikt om een andere standaard gebruikersnaam te specificeren voor een bepaalde host of hosts.

Escape karakter
Wanneer u een interactieve SSH login maakt, Kan het escape karakter worden gebruikt om de verbinding te verbreken en af te sluiten of op te schorten.

Comprimeer SSH verkeer?
Indien deze optie is aangezet, zal de SSH client alle data comprimeren die naar deze host word verstuurd met het gzip algoritme. Dit kan handig zijn indien u grote files kopieert met scp over een langzame verbinding.

Lokale poorten om door te sturen naar de server
In deze tabel kunt u locale poort nummers invullen die doorgestuurd moeten worden naar sommige host en poorten van de SSH server. Dit kan handig zijn indien uw enige toegang tot het netwerk via een SSH login loopt bij 1 machine op dat netwerk, en u ook nog toegang wilt hebben tot andere services zoals web of POP servers.

Server-poorten die lokaal doorgestuurd moeten worden
In deze tabel kunt u poort nummers invullen op de server die doorgestuurd moeten worden naar sommige host en poorten op de client machine's netwerk.
