Questa pagina mostra le opzioni per ogni singolo host, per gli host il quale nome corrisponde a qualche pattern, o per tutti gli host. Alcune delle opzioni meno ovvie sono :
Login come utente
Normalmente se nessun username è dato alla line di comando di ssh, il nome corrente dell' utente è passato come login a SSH. In ogni caso, questa opzione ti permette di definire un differente username per dei particolari host.

Carattere di escape
Quando è effettuato un login con SSH in modalità interattiva, il carattere di escape è usato per interrompere o sospendere la connessione.

Comprimere traffico SSH?
Se l' opzione è attivata il client SSH comprime tutto il traffico verso questo server con l' algoritmo gzip. Ciò può essere utile se stai copiando file con scp attraverso un collegamento lento.

Porte locali redirette al server
In questa tabella puoi inserire numeri di porte locali le quali saranno redirette sulle porte SSH di qualche altro server. Questo può essere utile se il tuo solo accesso a certe reti è permesso solo attraverso connessione SSH passanti per una macchina di questa rete e vuoi accedere ad altri servizi disponibili come web server o server POP.

Porte redirette localmente
In questa tabella puoi inserire numeri di porte locali le quali saranno redirette su host o porte della rete del client
