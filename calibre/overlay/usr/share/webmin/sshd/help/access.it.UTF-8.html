Controllo accesso
In base alla versione di SSH che stai usando potrai configurare,da questa pagina, uno o due tipi di controllo di accesso.

Puoi permettere o negare l' accesso al sistema grazie a una lista di hostname (come www.foo.com), indirizzi Ip oppure modelli (come *.foo.com). Questo tipo di controllo sugli accessi non è supportato da tutte le versioni di SSH.

Puoi permettere o negare l' accesso al sistema grazie ad una lista di account di utenti locali(come jcameron), modeli di account (come jcam*) o una combinazione di nomi utente e hostname (come jcameron@www.foo.com). L' accesso ai gruppi può essere implementato con una lista di nomi di gruppo o modelli ma non può includere hostname.
