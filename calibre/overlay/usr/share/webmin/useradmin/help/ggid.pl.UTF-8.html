Numer ID grupy
Podobnie do numeru ID użytkownika, numer ID dla grupy lub numer GID jest tym, czego system w rzeczywistości używa dla wymuszenia praw do plików i procesów. Jeśli dwie grupy maja ten sam numer GID, są one równoważne podczas sprawdzania uprawnień. Tworząc nową grupę Webmin wypełni to pole numerem GID, który nie jest używany przez inna grupę.
