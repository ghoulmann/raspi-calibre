Shell
Когда пользователь входит в систему по telnet или с консоли, или открывает окно xterm в оконной системе X, то программа называемая shell пользователя запускается для интерпретации вводимых пользователем команд. Если пользователь не имеет допустимого shell или имеет такой, который ничего не делает (например /bin/false), такой пользователь не сможет регистрироваться в системе. Это может быть полезным, если вы хотите создать пользователя имеющего возможность только отправлять и принимать почту.

В данном поле перечисленны все shellы используемые пользователями вашей системы. Вы можете выбрать один из них, или выбрать Другой.. и ввести полный путь к shell в текстовом поле ниже.

Все Unix системы поставляются с Bourne shell (/bin/sh) и C Shell (/bin/csh). Другие известные и легкие в использовании shellы tcsh, zsh и bash.
