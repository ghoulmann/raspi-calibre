Пароль
Когда пользователь входит в систему по telnet или с консоли, он в нормальном случае должен ввести пароль. Для пароля существуют четыре различные опции:
  • Спросить при первом входе
    Когда пользователь входит в систему первый раз по telnet или с консоли в текстовом режиме, система сама спросит пользователя ввести новый пароль. Не рекомендуем пользоваться данной опцией поскольку она не работает для многих других сервисов (в частности файлового сервера).

  • Не требуется пароля
    Пользователя может входить в систему без пароля. Будьте осторожны с этой опцией, опасайтесь нарушений безопасности.

  • Вход в систему не разрешен
    При выборе данной опции пользователь не сможет входить в систему вовсе, в том числе и для чтения почты по протоколу POP3

  • Зашифрованный пароль
    Пароли системы хранятся в односторонне зашифрованной форме. Эта опция позволяет Вам ввести пароль пользователя напрямую в зашифрованном виде. Данная опция может быть полезна в случае копирования паролей других пользователей.

  • Текстовый пароль
    Для установки пароля пользователя выберите этот пункт и введите пароль в текстовое поле. Данный пароль будет зашифрован и сохранен в системе.
