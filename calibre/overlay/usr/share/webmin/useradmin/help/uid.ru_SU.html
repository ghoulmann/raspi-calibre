ID пользователя
ID пользователя или UID - это числовой идентификатор, который система использует для проверки прав на файлы и процессы. Если два пользователя имеют одинаковый UID, то они имеют одинаковый доступ к файлам друг друга - таким образом каждый пользователь должен иметь уникальный UID. При создании нового пользователя данное поле заполняется автоматически UIDом не используемым другими пользователями.

Пользователь root всегда имеет UID = 0. Если вы создадите другого пользователя с UID 0, то этот пользователь будет иметь ту же силу, что и пользователь root.

Если при редактировании пользователя вы сменили UID, то опция Изменение ID пользователя на файлы? определяет какие файлы (если существуют) меняют своего владельца.
