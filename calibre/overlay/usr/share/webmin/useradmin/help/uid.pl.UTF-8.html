Numer ID użytkownika
Numer ID użytkownika lub UID jest tym, czego system w rzeczywistości używa dla wymuszenia praw do plików i procesów. Jeśli dwóch użytkowników ma ten sam numer UID, każdy z nich może uzyskać dostęp do plików drugiego - zatem każdy użytkownik powinien mieć unikalny numer UID. Podczas tworzenia użytkownika Webmin nada mu taki numer UID, jakiego nie używa żaden inny użytkownik.

Użytkownik root zawsze posiada UID równu 0. Jeśli utworzysz innego użytkownika z UID równym 0, ten użytkownik będzie równie potężny jak root.

Jeśli zmieniając istniejącego użytkownika zmienisz jego UID, opcja Zmienić ID użytkownika na plikach? pozwala ci określić które pliki (jeśli jakiekolwiek) będą nadal należały do tego użytkownika po zmianie UID.
