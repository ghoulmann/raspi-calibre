Överföringsmod
Sätter IDE-överföringsmod för nyare (E)IDE/ATA2-diskar.

Detta används oftast kombinerad med Använda DMA när DMA ska användas till/från en disk på ett interface-chipset som stöds.

Förutom detta används flaggan nästan aldrig eftersom (nästan) alla moderna IDE-diskar kör sin snabbaste PIO-överföringsmod som standard vid start. Det kan vara såväl onödigt som riskabelt att använda den.

Var mycket försiktig med att använda denna flagga!
