Soporte de Entrada/Salida de 32-bit
Configura el soporte de Entrada/Salida de 32 bits de (E)IDE.

Este valor funciona con casi cualquier chipset IDE de 32 bits, pero supone una ligera sobrecarga. Dese cuenta que "32 bits" hace referencia a las transferencias entre el bus PCI o VLB y la controladora IDE: todos los discos (E)IDE sólo disponen de una conexión de 16 bits sobre la faja hacia la controladora de discos.
