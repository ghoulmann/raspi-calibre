Hdparm
Aide à configurer plusieurs options des disque (E)IDE supporté par le pilote IDE/ST-506 de Linux.

Nécessite un noyau de version 1.2.13 ou plus récent. Certaines options ne pourrait pas fonctionner correctement sur de plus vieux noyau. De plus, certaines options nécessite un kernel plus récent avec le nouveau pilote (E)IDE, comme dans la version 2.0.

La colonne Disque(s) affiche tout les disques IDE. Cliquer sur un pour le configurer et éditer ou ajouter sa configuration dans le fichier de script.

État d'alimentation affiche l'état d'alimentation d'énergie du disque, que ce soit Normal, En attente quand le disque est en attente et la consommation électrique moindre ou Complètement arrêté où le moteur cesse complètement de tourner. display the state of the drive, if it's standby low power mode, drive has spun down, sleep mode lowest power mode, drive is completely shut down or active/idle normal operation.
Si le disque est dans un des modes d'économies d'énergie, vous n'avez qu'a cliquer sur l'état pour réveiller le disque et redémarrer le moteur.

Deux test de vitesse sont aussi fait pour voir d'éventuels fluctuations dans les peformances après des changements. Si le disque est dans un mode d'économie d'énergie ces tests sont ignoré.
