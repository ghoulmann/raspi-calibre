32-bit I/O ondersteuning
Zet de (E)IDE 32-bit I/O ondersteuning.

Deze waarde werkt met bijna alle 32-bit IDE chipset's, maar kan voor een klein beetje meer belasting zorgen. Let op dat "32-bit" alleen refereert aan data overdracht over de PCI of VLB bus naar de interface kaart : Alle (E)IDE harddisken hebben alleen maar een 16-bit verbinding over de kabel van de interface kaart.
