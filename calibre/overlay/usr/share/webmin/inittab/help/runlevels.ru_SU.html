Уровни выполнения
Данное поле может содержать несколко символов для разных уровней выполнения. Например 123 означает, что процесс должен стартовать в уровнях выполнения 1, 2, и 3.

При смене уровня выполнения любые процессы, которые не предназначены для нового уровня выполнения прибиваются сначала сигналом SIGTERM, а затем сигналом SIGKILL.
