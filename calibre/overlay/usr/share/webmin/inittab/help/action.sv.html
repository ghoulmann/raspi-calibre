Åtgärd
Talar om vilken åtgärd som ska vidtas.

Giltiga åtgärder är:

Fortsätt

Processen startas om så fort den av någon anledning avslutats (t ex getty).

Vänta

Processen startas när den angivna körnivån nås och init väntar på att den ska avslutas.

En gång på nivå

Processen kommer att köras en gång när den angivna körnivån nås.

Vid start

a Processen kommer att köras när systemet startas. Körnivåfältet ignoreras.

Vid start, vänta på avslutning

Processen kommer att köras när systemet startas och init väntar på att den ska avslutas (t ex /etc/rc). Körnivåfältet ignoreras.

Gör ingenting

Gör ingenting.

Alltid vid A, B, C

En process som markerats med en på begäran-körnivå kommer att köras varje gång den specificerade på begäran-körnivån anropas. Körnivån kommer dock inte att ändras (på begäran-körnivåer är 'a', 'b' och 'c').

På standardnivå

Initdefault-posten anger den körnivå som ska gälla vid systemstart. Om du inte angivit någon standardnivå kommer init att fråga efter körnivå via konsolen. Processfältet ignoreras.

Under systemstart

Processen kommer att köras under systemstart, före alla processer som angivits under Vid start och Vid start, vänta på avslutning. Körnivåfältet ignoreras.

Vid strömavbrott

Processen kommer att köras vid strömavbrott. Init får vanligen reda på detta via en process som kommunicerar med en UPS anslutet till datorn. Init väntar på att processen ska avslutas innan den fortsätter.

Vid strömavbrott, utan väntetid

Likadant som Vid strömavbrott utom att init inte väntar på att processen ska avslutas.

Då strömmen kommer tillbaka

Processen kommer att köras då init får veta att strömmen kommit tillbaka.

Vid UPS-svikt

Processen kommer att köras då init får veta att UPS-batteriet är nästan slut och spänningen kommer att försvinna (förutsatt att den externa UPSen och den övervakande processen kan upptäcka detta).

Ctrl-Alt-Del

Processen kommer att köras då init får signalen SIGINT. Detta betyder att någon vid systemkonsolen har tryckt CTRL-ALT-DEL. Vanligen vill man då köra någon slags shutdown, antingen för att gå över i single-user eller för att starta om maskinen.

Speciell tangentkombination

Processen kommer att köras då init tar emot en signal från tangentbordshanteraren som betyder att en speciell tangentkombination skrevs in på konsoltangentbordet.
