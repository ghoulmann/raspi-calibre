Proceso
Este campo contiene la línea de comando del programa que debería de ser ejecutado en el nivel de ejecución especificado (p.ej. /etc/init.d/rc, /sbin/getty...).
Si el campo de proceso comienza con un caracter '+', init no hará control de cuentas con utmp y wtmp para ese proceso. Esto es necesario para gettys que insisten en realizar su propio control de utmp y wtmp. Este es también un error histórico.
