Esegui nella directory
Per impostazione predefinita il comando specificato verrà eseguito nella directory che contiene il programma relativo a questo modulo Webmin (che dovrebbe essere /usr/libexec/webmin/custom). Questa opzione può essere usata per specificare una directory differente. Questo può essere utile, ad esempio, per evitare di utilizzare i percorsi completi per il comando e i suoi parametri.
