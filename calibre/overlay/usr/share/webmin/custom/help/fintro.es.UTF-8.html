También puede usar este módulo para configurar botones de edición de ficheros que edita con frecuencia. También puede ser útil si quiere conceder permiso de edición de ciertos ficheros a usuarios no privilegiados que de otra manera no podrían.

Como los botones de comandos, la página principal también mostrará un botón por cada editor de fichero definido. Para editar un editor existente, simplemente pulse el enlace Editar editor de fichero siguiente. Para crear un nuevo editor pulse el enlace Crear un nuevo editor de fichero al final de la página.
