Questo modulo può anche essere usato per impostare pulsanti per la modifica di file che vengono modificati di frequente oppure per permettere ad un utente non affidabile i diritti di modifica solo per pochi file.

Come i pulsanti relativi ai comandi la pagina principale del modulo visualizzerà un pulsante per ciascun file editor definito. Per modificare un editor esistente è sufficiente cliccare sul link Modifica il file editor che si trova sotto di esso. Per creare un nuovo editor clicca sul link Create un nuovo file editor che si trova in fondo alla pagina.
