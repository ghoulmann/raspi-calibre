Uit te voeren opdracht vooraf een bewerking
Iedere opdracht die hier word ingegeven word uitgevoerd als root voordat de inhoud van de file word getoond om te bewerken. Dit kan gebruikt worden om bijvoorbeeld een file te kopieeren van een ander systeem, en de Uit te voeren opdracht nadat de file is opgeslagen kan worden gebruikt om de file terug te kopieeren.
