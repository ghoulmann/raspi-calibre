Proprietario del file
Sono disponibili due opzioni.
Con la prima puoi decidere che il proprietario del file rimanga quello attuale.
Con la seconda, invece, puoi decidere di assegnare la proprietà del file ad un utente o gruppo specifico. La proprietà del file verrà modificata dopo il suo salvataggio.
Questa opzione è molto utile per i nuovi file creati. Infatti quando un file non esiste e viene creato tramite editor la proprietà del file stesso verrà assegnata all'utente e al gruppo Webmin che lo crea (solitamente root and root).
