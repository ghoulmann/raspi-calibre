Permessi sul file
Sono disponibili due opzioni.
Con la prima puoi decidere di lasciare i permessi allo stato attuale.
Con la seconda, invece, puoi inserire un numero ottale di 3 cifre che rappresentano i permessi da impostare sul file il suo salvataggio (il formato ottale dei diritti è lo stesso che si usa con il comando chmod)
Questa opzione è molto utile per i nuovi file creati. Infatti quando un file non esiste e viene creato tramite editor i permessi assegnati saranno quelli standard (solitamente 644).
