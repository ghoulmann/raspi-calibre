This module can also be used to setup buttons for editing frequently-edited files. This can also be useful if you want to grant some untrusted user the rights to edit only a few files.

Like the command buttons, the main page will also display one button for each defined file editor. To edit an existing editor, just click on the Edit file editor link below it. To create a new editor, click on the Create a new file editor link at the botton of the page.
