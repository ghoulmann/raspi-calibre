Gestor de Procesos - Vista por Usuario
Esta página muestra todos los procesos en ejecución agrupados por el usuario a quien pertenecen. Aquellos usuarios que tengan más de un proceso en ejecución los verán clasificados por uso de UCP bajo el nombre de cada usuario. Para cada proceso se muestra la PID, el uso de UCP y el comando. Se puede hacer click en la PID para mostrar más información.
