Che usa il filesystem
Questo criterio di ricerca permette di trovare quei processi che stanno utilizzando un file o una directory che si trova nel file system indicato. Se un processo ha aperto un file in lettura o in scrittura o si trova in una directory del file system indicato sarà incluso tra i risultati della ricerca.
Questo tipo di criterio può essere utile per trovare tutti quei processi che impediscono lo smontaggio di un determinato file system.
