Менеджер процессов - Форма поиска
Эта форма позволяет вам найти процессы отвечающие определенным критериям. При нажатии кнопки Найти, внизу страницы отобразится список найденных процессов. Для каждого из-них будут показаны его PID, владелец, использование CPU и команда. Для просмотра дополнительной информации о процессе, можно щелкнуть на его PID.

Ниже списка с результатами поиска, находятся кнопки отправки сигналов найденным процессам. Выберите из списка сигнал, который вы хотите отправить процессу, и нажмите кнопку Послать сигнал. Самые часто используемые сигналы это KILL и TERM, которые завершают запущенные процессы.
