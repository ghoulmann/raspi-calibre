Изменение процесса
Эта страница отображает подробности о запущенном процессе. В верхней части отображается следующая информация :
Команда
Программа и параметры командной строки для этого процесса
PID
Уникальный идентификатор процесса (Process ID)
Родительский процесс
Другой процесс, который создал этот. Кликните на родительском процессе, для отображения детальной информации о нём.
Владелец
Пользователь, которому принадлежит этот процесс, и с чьими правами он запущен
Использование CPU
Процент использования процессорного времени
Размер
Количество памяти занятое процессом. Некоторая часть из этой памяти, может использоваться совместно с другими процессами.
Использование времени CPU
Общее количество процессорного времени, использованного этим процессом с момента его запуска в минутах.
Приоритет (nice)
Приоритет, с которым запущен процесс. Подробнее...
Кроме того, в зависимости от установленной версии Unix может отображаться какая-либо дополнительная информация.

Ниже информации о процессе находится кнопка для отправки сигналов этому процессу. Выберите сигнал, который вы хотите отправить процессу и нажмите кнопку Послать сигнал. Самые полезные сигналы это KILL и TERM, которые завершают процесс.

Если у процесса есть какие-либо дочерние процессы, они отображаются внизу страницы. Нажмите на ID дочернего объекта, для отображения подробной информации.
