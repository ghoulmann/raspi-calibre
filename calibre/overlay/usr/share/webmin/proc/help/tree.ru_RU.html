Менеджер процессов - Отображение в виде дерева
Эта страница отображает список всех запущенных процессов с указанием их детей, которые находятся ниже родителей и имеют больший отступ в начале строки. Для каждого процесса можно увидеть его PID, владельца и команду. Для просмотра дополнительной информации о процессе, щелкните на его PID.
