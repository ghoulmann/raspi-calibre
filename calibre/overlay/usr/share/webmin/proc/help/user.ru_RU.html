Менеджер процессов - Пользователи
Эта страница отображает все запущенные процессы, сгруппированные по их владельцам. Процессы каждого пользователя отсортированы по степени использования CPU. Для каждого процесса можно увидеть его PID, использование CPU и команду. Для просмотра дополнительной информации о процессе, щелкните на его PID.
