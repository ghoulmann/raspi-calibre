Ändra process
På denna sida visas uppgifter om en aktiv process. Högst upp visas följande information:
Kommando
Program och kommandoradsargument för denna process.
Process-ID
Unikt process-ID.
Föräldraprocess
Processen som skapade denna process. Klicka på namnet för att se uppgifter om föräldraprocessen.
Användare
Den användare som äger processen och med vars rättigheter processen körs.
CPU-användning
Andel (i procent) av den totala CPU-tiden som denna process använder för tillfället.
Storlek
Mängd minne som används av denna process. Denna mängd kan delvis vara gemensam med andra processer.
Körtid
CPU-tid (i minuter) som processen totalt har använt sedan den startades. Om processen inte är väldigt CPU-intensiv ska detta inte vara lika mycket som den tid som gått sedan processen startades.
Nice-nivå
Prioritet som processen körs med. Det finns mer information om nice-nivå.
Det kan också finnas mer information beroende på vilken Unix-version du kör.

Nedanför processinformationen finns en knapp för att skicka en signal till processen. Välj från listan den signal du vill skicka och tryck sedan på Skicka signal. De mest användbara signalerna är KILL och TERM som slår ihjäl processen.

Om processen har några underprocesser visas dessa längst ned på sidan. Du kan klicka på ett process-ID för att se mer uppgifter om respektive process.
